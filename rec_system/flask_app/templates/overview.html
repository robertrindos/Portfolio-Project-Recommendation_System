Need movie recommendations?
Rob can help.

Utilizing Rotten Tomatoes data in combination with data science practices, allows this app to craft accurate movie suggestions. It finds movies with similar actors, directors, producers, genres, content ratings, and high reviews.

Built upon the Google Cloud Platform, the results are fast and the scalability is endless. The data pipelines within this cloud system are designed to clean, process, and manipulate data using natural language processing (NLP) practices.

For more info on how the recommendation system runs "under-the-hood", check out the README.txt file in the GitHub repository below.
